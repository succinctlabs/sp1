# sp1-gpu-prover

Complete GPU prover implementation for SP1.

Integrates all GPU proving components into a complete prover that can generate SP1 proofs using CUDA acceleration. This is the main entry point for GPU-accelerated proving.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
