# sp1-gpu-jagged-sumcheck

GPU-accelerated sumcheck protocol for jagged polynomials.

Implements the sumcheck protocol on GPU for jagged (variable-sized) polynomial structures, enabling efficient proving over the non-uniform table sizes in SP1's arithmetization.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
