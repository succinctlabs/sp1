# sp1-gpu-shard-prover

GPU shard proving orchestration.

Orchestrates the proving of individual shards on GPU, coordinating trace generation, commitment, and proof generation for each shard in the SP1 proving pipeline.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
