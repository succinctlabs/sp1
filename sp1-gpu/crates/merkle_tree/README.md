# sp1-gpu-merkle-tree

GPU-accelerated Merkle tree commitment scheme.

Implements Merkle tree construction and commitment operations on GPU using Poseidon2 hashing, providing fast polynomial commitments for the SP1 proving pipeline.

---

Part of [SP1-GPU](https://github.com/succinctlabs/sp1/tree/dev/sp1-gpu), the GPU-accelerated prover for SP1.
