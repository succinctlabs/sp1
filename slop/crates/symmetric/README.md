# slop-symmetric

Re-exports [`p3_symmetric`](https://crates.io/crates/p3_symmetric) from [Plonky3](https://github.com/Plonky3/Plonky3) for use in the SLOP library.

This crate provides symmetric cryptographic primitives including compression functions and sponge constructions. These are used for building Merkle trees and other hash-based commitment schemes in proof systems.
